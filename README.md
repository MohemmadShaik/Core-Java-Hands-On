# Data types in Java

 Primitive Data Types : Total we have 8 Primitive data types in Java

 byte    --> 1 byte  
 short   --> 2 bytes  
 int     --> 4 bytes  
 long    --> 8 bytes  
 float   --> 4 bytes  
 double  --> 8 bytes  
 boolean --> 1 byte  
 char    --> 1 byte  

byte, short, int and long used for whole numbers
